WebLens Docs

Quick integration and environment reference.

Environment Variables

OPENAI_API_KEY=your_openai_key OPENAI_MODEL=gpt-4.1-mini FIRECRAWL_API_KEY=your_firecrawl_key PORT=8787

If `OPENAI_API_KEY` is missing, WebLens still works with heuristic fallback analysis.

API Endpoints

POST /api/analyze

{ "url": "https://example.com" }

Returns scraped content, AI analysis, and clone blueprint.

POST /api/ask

{ "question": "How does this site make money?", "context": { "analysis": {}, "blueprint": {}, "scraped": {} } }

Returns a contextual follow-up answer for the analyzed website.

Workflow

  1. User submits URL
  2. WebLens scrapes page HTML (Firecrawl optional fallback path)
  3. Signals are extracted (content + structure + tech hints)
  4. AI/heuristic reasoning produces strategy cards
  5. User can ask follow-up questions or copy blueprint
